Type&Count

Let's talk about the project...

Discription:

This is a word typing game consists of three levels easy, medium and difficult. Words are fetched from a WordAPI using fetch javascript function. For each level a definite time period is given in which the player has to write the displayed word. If player failed to do so they have to rewrite the word correctly to restart. It is a beginner level project developed solely for the purpose to practice javascript as well as improve my typing speed.

Skills learned:

HTML, CSS, Bootstrap, Javascript
