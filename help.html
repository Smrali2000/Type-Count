Type&Count

Rules of the game

  • This a word typing game with definite time limit.
  • There would be a word displayed and player has to write the same word in the space given.
  • There are three levels in this game. You can choose the levels as per your choice.
    Level Time
    Easy 7
    Medium 5
    Difficult 3
  • Complete writing the word in the given time otherwise game will be over.
  • In case the game is over, write the current word correctly to restart.
  • You will be rewarded with 1 point for every correct written word. In case of game over, when you restart the game score will be reset to 0.
